## Why Phake MCP?

`@phake/mcp` is purpose-built for deploying MCP servers on Cloudflare Workers, with full Node.js support for local development and self-hosted environments.

## Get started

<Steps>
  <Step title="Install the package">
    ```bash
    npm install @phake/mcp
    ```
  </Step>
  <Step title="Define your tools">
    ```typescript
    import { z } from "zod";
    import { defineTool } from "@phake/mcp";

    const greetTool = defineTool({
      name: "greet",
      description: "Returns a greeting for the given name",
      inputSchema: z.object({
        name: z.string().describe("Name to greet"),
      }),
      handler: async (args) => {
        return { message: `Hello, ${args.name}!` };
      },
    });
    ```
  </Step>
  <Step title="Create and export your server">
    ```typescript
    import { createMCPServer } from "@phake/mcp";

    const server = createMCPServer({
      adapter: "worker",
      tools: [greetTool],
    });

    export default server;
    ```
  </Step>
  <Step title="Deploy to Cloudflare Workers">
    ```bash
    wrangler deploy
    ```
  </Step>
</Steps>

<Note>
  Need to set up KV storage and encryption keys before deploying? Follow the [Cloudflare Workers deployment guide](/guides/cloudflare-workers) for full configuration instructions.
</Note>
